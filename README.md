# The goal is to build a little self driving car.

The current project is in the researching stage...

## Materials

1.- Nvidia Jetson Nano developer board
2.- Jetson Nano Adapter
3.- Camera Module
4.- WiFi Card
5.- Antenas

6.- Exceed magnet rc car chasis
